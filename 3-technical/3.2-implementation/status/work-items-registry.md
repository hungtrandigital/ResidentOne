## How to Use This Registry

1. **When creating a new work item:**
   - Add it to the appropriate section above
   - Include all required metadata (ID, title, domain, status, owner, dates)
   - Link to parent epic/slice via `parent_id`

2. **When updating a work item:**
   - Update status, dates, and related items
   - Move to "Completed" section when done

3. **When searching for work items:**
   - Use this registry to find work items by ID, domain, or status
   - Navigate to detailed documentation via links in this registry
