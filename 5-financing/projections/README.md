## Framework Applied

1. **SaaS Unit Economics Model**
   - CAC, LTV, LTV/CAC ratios, payback periods
   - Industry benchmarks validation

2. **Scenario Planning**
   - Conservative, Realistic, Optimistic scenarios
   - Sensitivity analysis

3. **Validation Framework**
   - Pilot building metrics tracking
   - Monthly/quarterly updates

## Key Changes from Original

- **CAC:** VND 8-15M (revised from VND 1-2M)
- **LTV:** VND 80-120M (revised from VND 168M)
- **LTV/CAC:** 5-10× (revised from 84-168×)
- **Year 1 Revenue:** VND 840M-1.2B (revised from VND 1.62B)
- **Year 1 Buildings:** 20-40 (revised from 50)
- **Break-Even:** Month 10-24 (revised from Month 7-8)
- **Year 1 Funding:** VND 500M-1.5B (revised from VND 200M-500M)

## Usage

- **For Planning:** Use Realistic Scenario as base case
- **For Fundraising:** Present all three scenarios
- **For Validation:** Follow pilot validation plan
- **For Updates:** Review quarterly or after major milestones
