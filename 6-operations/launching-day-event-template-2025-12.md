This document provides a comprehensive template and checklist for Launching Day events, based on real experience from pilot event:

## 1. Event Overview

### 1.1 Event Purpose

**Primary Goals:**
1. **Introduce Platform:** Demonstrate value proposition to residents and sellers
2. **Onboard Users:** Register residents and sellers on-site
3. **Build Community:** Create sense of community and engagement
4. **Generate Momentum:** Create word-of-mouth and early adoption

**Target Audience:**
- **Residents:** 100-200 residents (target: 100+ on launch day)
- **Sellers (Tiểu thương):** 30-50 sellers (target: 40+ on launch day)
- **BQL/BQT:** Building management team

### 1.2 Event Format

**Type:** Physical Event (In-Person)  
**Venue:** Building common area (lobby, community room, or outdoor space)  
**Duration:** 2-4 hours (afternoon/evening, e.g., 2-6 PM or 5-9 PM)  
**Date:** Day 32-34 of onboarding process (during Phase 6: Launch)

**Event Structure:**
1. **Opening & Introduction** (15-20 min)
2. **Platform Demo** (30-45 min)
3. **Onboarding Session** (60-90 min)
4. **Networking & Refreshments** (30-45 min)
5. **Closing & Next Steps** (15-20 min)

## 2. Cost Breakdown (Actual Experience)

### 2.1 Actual Cost Structure

**Total Cost: VND 35-40M per building** (based on real experience)

**Cost Breakdown:**

1. **Venue & Setup** (VND 8-12M)
   - Table & chair rental: VND 3-5M
   - Backdrop/banner production: VND 2-3M
   - Sound system rental (if needed): VND 1-2M
   - Decoration & setup: VND 2-2M

2. **Marketing Materials** (VND 3-5M)
   - Flyers and posters: VND 1-1.5M
   - Banners and signage: VND 1-2M
   - Platform demo materials: VND 0.5-1M
   - Registration materials: VND 0.5-0.5M

3. **Gifts & Incentives** (VND 15-18M) ⭐ **KEY COST DRIVER**
   - Participant gifts: VND 10-12M (100-150 participants × VND 80K-100K average)
   - Early adopter rewards: VND 3-4M (for first 20-30 sellers)
   - Lucky draw prizes: VND 2-2M (optional but recommended)

4. **Refreshments** (VND 3-5M)
   - Light refreshments (snacks, drinks): VND 2-3M
   - Optional: Simple meal (if evening event): VND 1-2M

5. **Event Support** (VND 3-4M)
   - Event coordinator time: VND 1-1.5M
   - Platform team support: VND 1-1.5M
   - Onboarding support staff: VND 1-1M

6. **Technology Setup** (VND 2-3M)
   - Projector/screen rental: VND 0.5-1M
   - WiFi setup/booster: VND 0.5-1M
   - Registration tablets/devices: VND 0.5-0.5M
   - Backup equipment: VND 0.5-0.5M

7. **Contingency** (VND 1-2M)
   - Unexpected costs
   - Buffer for variations

**Total: VND 35-40M per building**

---

### 2.2 Cost Optimization Opportunities

**Ways to Reduce Cost (Future Events):**
1. **Use Building Common Area:** Free venue (saves VND 3-5M)
2. **Borrow Equipment:** Borrow tables/chairs from building or partners (saves VND 2-3M)
3. **Optimize Gifts:** Focus on high-value, lower-cost gifts (saves VND 3-5M)
4. **Standardize Materials:** Reuse banners/materials across buildings (saves VND 1-2M)
5. **Volunteer Support:** Use community ambassadors for support (saves VND 1-2M)

**Optimized Cost Target:** VND 25-30M per building (after optimization)

### 4.2 Event Execution (2-4 Hours)

**Segment 1: Opening & Introduction (15-20 min)**

**Tasks:**
- [ ] Welcome participants
- [ ] Introduce BQL/BQT representatives
- [ ] Introduce platform team
- [ ] Explain event agenda
- [ ] Set expectations

**Key Messages:**
- Platform value proposition
- Resident-owned concept
- Trust & safety features
- Community building

**Success Criteria:**
- 80%+ participants engaged
- Clear understanding of platform purpose

---

**Segment 2: Platform Demo (30-45 min)**

**Tasks:**
- [ ] Live platform walkthrough
- [ ] Demonstrate key features:
  - [ ] User registration
  - [ ] Product listing (for sellers)
  - [ ] Search and discovery (for buyers)
  - [ ] Messaging system
  - [ ] Trust & safety features
- [ ] Q&A session

**Key Messages:**
- How to use platform
- Key features and benefits
- Trust & safety assurance
- Resident ownership

**Success Criteria:**
- 70%+ participants understand platform
- 50%+ participants interested in using

---

**Segment 3: Onboarding Session (60-90 min)**

**Tasks:**
- [ ] Help residents register
  - [ ] Set up registration stations
  - [ ] Guide registration process
  - [ ] Verify phone numbers
  - [ ] Help with first login
- [ ] Help sellers create first listings
  - [ ] Guide listing creation
  - [ ] Help upload photos
  - [ ] Help set prices
  - [ ] Verify seller accounts
- [ ] Answer questions
- [ ] Provide tutorials

**Targets:**
- 50-100 residents registered on-site
- 30-50 sellers onboarded on-site
- 20-30 first listings created

**Success Criteria:**
- 50+ residents registered
- 30+ sellers onboarded
- 20+ listings created

**Segment 4: Networking & Refreshments (30-45 min)**

**Tasks:**

- [ ] Facilitate networking

  - [ ] Lucky draw prizes (optional)

- [ ] Answer additional questions

**Success Criteria:**
- 80%+ participants receive gifts
- Positive feedback collected
- Networking happening

**Segment 5: Closing & Next Steps (15-20 min)**

- [ ] Thank participants
- [ ] Announce next steps

**Success Criteria:**
