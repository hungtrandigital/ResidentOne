# Audit Reports

This directory contains all audit reports organized by type.

## Structure

### File Placement Audits
- **[file-placement/](file-placement/)** - File placement audits (checking if files are in correct locations)

### Documentation Audits
- **[documentation/](documentation/)** - Documentation location and organization audits

### Placement Audits
- **[placement/](placement/)** - Specific placement audits for directories or topics (e.g., 1-ideas placement audit)

## File Naming Convention

- **File Placement Audits:** `file-placement-audit-YYYY-MM-DD.md`
- **Documentation Audits:** `documentation-location-audit-YYYY-MM.md` or `documentation-audit-[topic]-YYYY-MM-DD.md`
- **Placement Audits:** `placement-audit-[topic]-YYYY-MM-DD.md`

## When to Create Audit Reports

- After major documentation reorganization
- When @docs-guardian is called
- Periodically (weekly/monthly) for maintenance
- Before committing large changes
