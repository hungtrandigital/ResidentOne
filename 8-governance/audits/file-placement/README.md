# File Placement Audits

This directory contains file placement audit reports generated by the audit script.

## Purpose

File placement audits verify that all files are in correct locations according to the repository structure defined in `INDEX.md`.

## Audit Reports

- `file-placement-audit-2025-12-12.md` - Audit report from 2025-12-12
- `file-placement-audit-2025-12-13.md` - Audit report from 2025-12-13

## How to Run Audit

```bash
./8-governance/scripts/file-placement-audit.sh
```

The script generates a new audit report in this directory with the format: `file-placement-audit-YYYY-MM-DD.md`

## Understanding Results

- **✅ PASS:** 0 violations, 0 warnings
- **⚠️ WARNINGS:** 0 violations, > 0 warnings (review and fix if needed)
- **❌ FAIL:** > 0 violations (MUST FIX before proceeding)
