# Placement Audits

This directory contains specific placement audits for directories or topics.

## Purpose

Placement audits verify that files in specific directories or topics are correctly organized and placed.

## Audit Reports

- `placement-audit-1-ideas-2025-12-13.md` - Placement audit for 1-ideas directory
- `placement-verification-8-governance-2025-12-13.md` - Placement verification for 8-governance directory

## When to Create Placement Audits

- After reorganizing a specific directory
- When verifying file placement in a specific area
- After major structural changes to a directory

## File Naming

Format: `placement-audit-[topic]-YYYY-MM-DD.md`

Examples:
- `placement-audit-1-ideas-2025-12-13.md`
- `placement-audit-6-operations-2025-12-13.md`
