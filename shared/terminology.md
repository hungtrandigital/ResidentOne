# Terminology Guide — ResidentOne Platform

**Version:** 1.0  
**Last Updated:** 2025-12-12  
**Status:** Mandatory Reference

This guide standardizes terminology usage across all documentation to ensure consistency and clarity.

---

## Language Usage Rules

### Primary Language: English
- **All documentation should be in English** unless explicitly required otherwise
- **Vietnamese terms** may be used when:
  1. There is no direct English equivalent
  2. The term is a proper noun (e.g., building names, company names)
  3. The term is commonly used in Vietnamese business context and needs to be preserved

### Format for Vietnamese Terms
- **First use:** Vietnamese term (English translation) - e.g., "BQL (Building Management Board)"
- **Subsequent uses:** Use English translation or abbreviation
- **In tables/formal documents:** Use English translation with Vietnamese in parentheses if needed

## Abbreviations

### Standard Abbreviations
- **BQL** - Building Management Board (Ban Quản Lý)
- **BQT** - Building Management Committee (Ban Quản Trị)
- **SSO** - Single Sign-On
- **MVP** - Minimum Viable Product
- **GTM** - Go-to-Market
- **CAC** - Customer Acquisition Cost
- **LTV** - Lifetime Value

### Usage Rules for Abbreviations
1. **First use:** Full term (Abbreviation) - e.g., "Building Management Board (BQL)"
2. **Subsequent uses:** Abbreviation is acceptable
3. **In titles/headings:** Prefer full term unless abbreviation is very common

## Currency & Numbers

### Currency
- **Primary:** VND (Vietnamese Dong)
- **Format:** VND 500M, VND 1B (M = million, B = billion)
- **Alternative:** USD when specified (e.g., "USD 20,000")

### Numbers
- Use Western number format: 1,000,000 (not 1.000.000)
- Use abbreviations: 1M (million), 1B (billion), 1K (thousand)

## Document-Specific Guidelines

### Executive Summaries
- **Language:** English
- **Vietnamese terms:** Include translation on first use
- **Format:** "Vietnamese Term (English Translation)"

### Market Research Reports
- **Language:** English
- **Local terms:** May include Vietnamese for context, but always provide English translation
- **Data sources:** Vietnamese sources should be translated to English

### Technical Documentation
- **Language:** English only
- **Code/comments:** English only
- **Variable names:** English only

### Marketing Materials
- **Language:** Depends on target audience
- **Vietnamese materials:** Use Vietnamese
- **English materials:** Use English
- **Bilingual:** Clearly separate sections
